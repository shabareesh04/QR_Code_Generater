# 📱 QR Code Generator

## 📖 Description

A simple and responsive **QR Code Generator web app** built using HTML, CSS, and JavaScript.
It allows users to generate QR codes for any text or URL and download them instantly.

---

## 🚀 Features

* Generate QR codes for text or URLs
* Multiple size options (100x100 to 400x400)
* Download QR code as PNG image
* Responsive and user-friendly UI
* Input validation

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* QRCode.js Library

```bash
git clone https://github.com/your-username/qr-code-generator.git
```

### 2. Navigate to project folder

```bash
cd qr-code-generator
```

### 3. Open in browser

Just open `index.html` in your browser

## ▶️ Usage

1. Enter text or URL
2. Select QR code size
3. Click **Generate**
4. Click **Download** to save QR code

## 💡 How It Works

* Takes user input (text/URL)
* Uses **QRCode.js library** to generate QR
* Displays QR code dynamically
* Allows download as image
